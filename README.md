# Angular + Electron Seed

Base project to build your awesome Electron-based Angular app.

* Angular 1.5
* Angular UI Router
* Electron
* Gulp
* SASS

### Getting started

#### Clone repo
```
$ git clone https://github.com/JuanjoSalvador/angular-electron-seed.git
```

#### Install dependencies
```
$ npm install
$ bower install
```
#### Run it!
```
$ npm start
```
